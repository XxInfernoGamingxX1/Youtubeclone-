# 🔥 Lunify

> modern, ad-free, account-free video streaming web app powered by YouTube.

![Lunify Banner](https://via.placeholder.com/1200x300?text=Lunify)

---

## 🚀 About

**Lunify** is a sleek, fast, and minimal YouTube-based streaming app that lets you:

- 🎬 Watch videos instantly  
- 🔍 Search anything  
- 🚫 No login required  
- ⚡ No cluttered UI  
- 🎨 Clean modern design  

Built for speed, simplicity, and vibes.

---

## ✨ Features

- 🏠 **Home / FYP Page** (recommended videos)
- 🔥 **Trending Section**
- 🔍 **Search (Enter to search)**
- 🎬 **Instant Video Playback**
- 📱 **Fully Responsive (Mobile + Desktop)**
- ⚡ **Fast + Lightweight**
- 🔒 **Secure API (backend powered)**

---

## 🧠 Tech Stack

### Frontend
- HTML, CSS, JavaScript  
- Hosted on **Vercel**

### Backend
- Node.js + Express  
- Hosted on **Render**

### API
- YouTube Data API v3
